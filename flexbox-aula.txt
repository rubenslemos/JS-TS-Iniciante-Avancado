Flex box

Na primeira parte vimos os alinhamentos do flex box do justify-content e do align-items

Quando usamos o flex-direction row o justify-content altera o eixo x e o align-items o eixo y pois o eixo x nesse caso é o principal

Quando usamos o flex-direction Colum o justify-content altera o eixo y e o align-items o eixo x pois o eixo y nesse caso é o principal

podemos usar alguns direcionamentos:
flex-start = posiciona os elementos no inicio
flex-end = posiciona os elementos no fim
space around = posiciona os elementos com o espaçamento entre eles iguais
space between = divide igualmente o espaço para os elementos
stretch = estica os elementos para ocuparem todo espaço disponível no eixo
